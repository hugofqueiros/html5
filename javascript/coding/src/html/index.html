Gulp ES6 Boilerplate

Why Gulp ES6 Boilerplate?

This is a opinionated boilerplate, written in ECMA6, designed for web development. Employs Gulp to automate your everyday tasks live reloading development server, css compiling, image optimization, js, css and html minification and build/distribution website generation

Features

  1. Gulp - automated development, every task is written in ES6
    1. Optimization
    2. Minification
    3. Watch changes
    4. Build development and production environments
  2. BrowserSync development
    1. Live reloading on change
    2. Inject CSS changes - won't reload page
  3. Styles
    1. Less - - CSS pre-processor (you can easily modify it to SASS, replacing the gulp less plugin for sass)
    2. PostCSS - plugins autoprefixer and minify
  4. Test
    1. Basic Karma configuration
